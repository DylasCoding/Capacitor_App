
## 7. Meme Maker
- **Mục tiêu:** Tạo meme từ ảnh thư viện, thêm chữ và chia sẻ.
- **Plugin:** `@capacitor/camera`, `@capacitor/filesystem`, `@capacitor/share`.
- **Yêu cầu tối thiểu:**
    - Chọn ảnh từ thư viện.
    - Vẽ chữ (canvas) lên ảnh.
    - Lưu và chia sẻ meme.
- **Mở rộng:**
    - Thêm bộ lọc ảnh.
    - Khung hình preset cho mạng xã hội, khung tròn thì bo ảnh tròn, khung vuông thì bo ảnh vuông.
    - Thêm sticker, biểu tượng cảm xúc.
